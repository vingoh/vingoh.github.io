# Agent 学习路径

基于Hugging Face的[Agent Course](https://github.com/huggingface/agents-course)以及datawhale的[Hello Agent](https://github.com/datawhalechina/hello-agents)两个项目，以及总结最近看到的一些其他课程，总结出如下的学习路径。
鉴于之前有一定的深度学习基础，因此跳过了一些对深度学习的入门介绍。希望做到的是在每个关键点学习完成之后，总结自己的学习记录，输出想法，而非简单的复制粘贴。

对应的agent项目由GPT总结后给出推荐，在后续过程中持续优化对应项目与内容，以符合实际需求为第一要义。


---

## Phase 0：LLM 基础

### 需要掌握的内容

- Transformer 基本结构  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter3/第三章%20大语言模型基础.md  

- 大模型训练范式（Pretraining / Instruction Tuning / RLHF）  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter3/第三章%20大语言模型基础.md  

- In-context learning（zero-shot / few-shot / CoT）  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter3/第三章%20大语言模型基础.md  

- Token 与上下文窗口  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit1/llm_basics.md  

- LLM 推理机制（next-token prediction / sampling）  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit1/llm_basics.md  

- Embedding 基础  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter8/第八章%20记忆与检索.md 

### 补充说明

这一阶段重点是理解 LLM 的能力边界，以及为什么它需要 Agent 结构来补足能力。

---

## Phase 1：Agent 最小闭环

### 需要掌握的内容

- Agent 基本结构（LLM + Tools + Loop）  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit1/agents.md  

- ReAct 模式  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter4/第四章%20智能体的思维链与推理.md  

- Tool Calling  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit1/tools.md  

- Tool 设计（name / description / schema）  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit2/tools.md  

- RAG（Retrieval-Augmented Generation）  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter8/第八章%20记忆与检索.md

- Agent 框架（LangChain / LlamaIndex / LangGraph）  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit2/frameworks.md  

### 需要完成的项目

- 代码库问答 Agent
  - 支持搜索代码
  - 支持读取文件
  - 支持回答结构性问题

### 补充说明

重点是实现“动态决策是否调用工具”，而不是简单检索问答。

---

## Phase 2：Agent Workflow（多步执行）

### 需要掌握的内容

- 多步推理（multi-step reasoning）  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit1/agents.md  

- Agent Workflow（Plan → Execute → Reflect）  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter4/第四章%20智能体的思维链与推理.md  

- Agent Loop  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter6/第六章%20框架开发实践.md

- Workflow / Graph（LangGraph）  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit2/frameworks.md  

### 需要完成的项目

- Debug Agent
  - 输入错误日志
  - 搜索相关代码
  - 分析原因
  - 给出修改建议

### 补充说明

这一阶段的核心是让 Agent 具备“连续行动能力”。

---

## Phase 3：Agent 系统设计

### 需要掌握的内容

- Context Engineering  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter9/第九章%20上下文工程.md 

- Memory 系统  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter8/第八章%20记忆与检索.md  

- Planning 机制  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter4/第四章%20智能体的思维链与推理.md  

- Tool 设计（系统级）  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit2/tools.md  

- Agent 评估  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter12/第十二章%20智能体性能评估.md   

### 需要完成的项目

- 代码修改 Agent
  - 理解需求
  - 定位代码
  - 修改逻辑
  - 输出解释

### 补充说明

这一阶段开始真正拉开与普通“调用API”的差距。

---

## Phase 4：工程化 Agent

### 需要掌握的内容

- Agent 系统设计（orchestration）  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter6/Chapter6-Framework-Development-Practice.md  

- 多 Agent 系统  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit2/smolagents/multi_agent_systems.mdx 

- 性能优化（latency / cost）  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit3/performance.md  

- 数据闭环与优化  
  - 对应学习：  
    - https://github.com/huggingface/agents-course/blob/main/units/zh-CN/unit3/evaluation.md  

- Evaluation 系统  
  - 对应学习：  
    - https://github.com/datawhalechina/hello-agents/blob/main/docs/chapter12/第十二章%20智能体性能评估.md  

### 需要完成的项目

- C++ 工程 Agent 系统
  - 代码理解
  - debug
  - 自动修改
  - PR 生成

### 补充说明

目标是具备完整系统设计能力，而不是单点能力。

---

## 最终能力目标

- 能理解 Agent 的完整结构（LLM + Tools + Loop + Memory + Planning）
- 能设计针对代码工程的 Agent 系统
- 能实现多步执行与工具协作
- 能控制上下文、成本与准确率
- 能完成工程级项目并具备面试竞争力
